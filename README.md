# 🎓 돌아봄 (Dorabom)

> 초등학교 6학년 졸업생을 위한 추억 회상 및 감사 앱

## 소개

**버킷리스트**가 *앞으로 하고 싶은 것*이라면,
**돌아봄**은 *지나온 시간에 대한 감사*입니다.

초등학교 6년간의 의미 있는 순간들을 돌아보고, 함께했던 사람들에게 감사의 마음을 전할 수 있는 웹앱입니다.

## 주요 기능

### 📝 5가지 카테고리 회상
- ⭐ **의미 있던 순간** - 뿌듯했던 일, 성장의 경험
- 📸 **소중한 추억** - 친구들과의 재미있던 일
- 💝 **고마웠던 사람** - 도움을 준 친구, 선생님, 가족
- 🌟 **내가 좋아했던 것** - 즐거웠던 수업, 장소, 활동
- 🌱 **앞으로의 다짐** - 중학생이 되어서도 간직하고 싶은 것

### 💌 감사 편지 작성
가장 고마운 사람에게 마음을 담은 편지를 씁니다.

### 💾 저장 기능
- PNG 이미지로 저장
- PDF 문서로 저장

## 사용 방법

1. 이름을 입력합니다
2. 5개의 카테고리를 하나씩 채워갑니다
3. 모든 카테고리를 완료하면 감사 편지를 작성합니다
4. 완성된 결과물을 이미지나 PDF로 저장합니다

## 설치 및 실행

### 로컬 실행
```bash
# 폴더로 이동
cd dorabom

# 간단한 웹 서버로 실행
python -m http.server 8000
# 또는
npx serve
```

브라우저에서 `http://localhost:8000` 접속

### GitHub Pages 배포
1. GitHub에 저장소 생성
2. 파일 업로드
3. Settings → Pages → Source를 main 브랜치로 설정
4. `https://[username].github.io/dorabom` 에서 접속

## 기술 스택

- HTML5 / CSS3 / JavaScript (Vanilla)
- [html2canvas](https://html2canvas.hertzen.com/) - 이미지 캡처
- [jsPDF](https://github.com/parallax/jsPDF) - PDF 생성
- LocalStorage - 데이터 저장

## 파일 구조

dorabom/

## 활용 예시

- **도덕과 연계**: 감사, 배려, 관계 등 핵심 가치 내면화
- **정서 발달**: 긍정적 정서 경험과 회복탄력성 강화
- **자기이해**: 나에게 의미 있는 것이 무엇인지 성찰
- **표현력**: 추상적 감정을 구체적 언어로 표현
